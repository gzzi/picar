# The raspberry pi custom hat

## Pinout


Pin 19 / BCM 10 / wiringpi 12 = LED

Pin 32 / BCM 12 / wiringpi 26 = PWM0
Pin 33 / BCM 13 / wiringpi 23 = PWM1
